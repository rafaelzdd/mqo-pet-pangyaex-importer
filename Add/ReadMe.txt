各ツールの最終版と新たに作成したツールです。

変更点
MpetMqo.exe
　各頂点ごとのボーンに対する重み係数を「blend」オブジェクトとして作成

AjstMpet.exe
　上記MpetMqo.exeで作成するようにした「blend」オブジェクトに対応

AddMpet.exe
　オブジェクトを追加するとクライアントで読み込み失敗するmpetがあったのを対処


新規ツール
CnctMpet.exe
　使い方
　CnctMpet.exe mpet1 mpet2
　mpet1にmpet2を接続した*.newを作成
　リネームしただけではパーツが欠けてしまうエリカ下半身やクー上半身などに使用

FitMpet.exe
　使い方
　FitMpet.exe mpet1 mpet2
　mpet1の各頂点を、mpet2の十分に近い頂点座標に合わせた*.newを作成
　mpet<->mqoを繰り返すと計算誤差等で他パーツとの接続部分に微妙な隙間が
　出来る場合があるので、それを修正するのに使用

※新規ツールはパーツが別れていないpetファイルには不要なので非対応


blendオブジェクトについて
　AddMpet.exeで追加した頂点は所属ボーンに対する重み係数が必ず1.0（100%）に
　なりますが、blendオブジェクトを弄ってAjstMpet.exeに渡すことで滑らかな変形が
　可能です。
　MpetMqo.exeは各頂点ごとに以下の４つの頂点とそれを結んだ２つの三角面を
　作成します。

　頂点１：（X1, Y1, Z1） = 対象の頂点座標
　頂点２：（X2, Y2, Z2） = （X1, Y1-10.0, 所属するボーン番号）
　頂点３：（X3, Y3, Z3） = （X1, Y1-10.0, MatrixBlend対象ボーン番号）
　頂点４：（X4, Y4, Z4） = （X1, Y1-20.0, Z2+MatrixBlend対象ボーンに対する重み係数）
　１-２-３平面
　２-４-３平面

　２つの三角面は所属ボーンと同名の材質が設定されますが、材質が設定されてい
　ない場所（白、不透明）は基本的に弄ってはいけません。読み込み時に固まります。

　例）座標(0.0, 1.0, 2.0)の頂点をボーン33とボーン44の重み係数50%に設定する場合

　頂点１：（0.0, 1.0, 2.0）
　頂点２：（0.0, -9.0, [33.0]）
　頂点３：（0.0, -9.0, [44.0]）
　頂点４：（0.0, -19.0, [33.5]）

　実際に意味があるのは[ ]の部分のみです。

　ある程度３D関係の知識が必要なので上記以上に説明するつもりはありません。
　判る人のみ弄ってください。意味不明のまま適当に弄ると読み込み時に固まります。
